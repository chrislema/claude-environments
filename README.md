# Claude Environments

Claude environment for software delivery work.

This project packages software development expertise into Claude-native parts:

- `CLAUDE.md` for the standing constitution
- `agents/` for role-specific subagents
- `commands/` for slash-command workflows
- `skills/` for reusable implementation and review playbooks
- `templates/` for consistent outputs and handoffs

The goal is not to reproduce orchestration runtime mechanics. The goal is to preserve durable engineering judgment and make it easy to invoke inside Claude's environment.
